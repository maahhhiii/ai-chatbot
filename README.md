# 🤖 Gemini AI Chatbot

A Streamlit-based AI chatbot powered by Google Gemini.

## Features

- Multi-chat support
- Chat history stored in SQLite
- PDF export
- Delete chats
- Automatic chat titles
- Modern Streamlit UI

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- SQLite
- SQLAlchemy

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
